# Bandmate Club

This repository is a monorepo for [BandMate](http://bandmate.club)

## Technologies

### Payment Server

- TypeScript 3.7
- Postgresql
- Apollo GraphQL
- ExpressJS
- InversifyJS

### Payment Frontend

- React
- Apollo GraphQL
- Ant Design

### Common

- TravisCI integration
- Deployable to Heroku
